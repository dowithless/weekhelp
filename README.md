# weekhelp README

这是一个 VsCode 插件，一个简单的周报助手，专为开发者设计。

## Features

1 个被动技能：自动收集 git commit 的提交信息，存储到以当前周命名的 Markdown 文件中。

2 个主动技能：

- Open Weekhelp File 查看/编辑 当前周的 Markdown 文件
- Open Weekhelp Folder 打开 weekhelp 文件夹，这是存储全部 Markdown 文件的的文件夹。
